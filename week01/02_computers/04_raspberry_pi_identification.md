| Component | Function |
|-----------|----------|
| SoC |The main "brain" of the device. It contains the processor and other key circuits that handle computing tasks. |
| GPIO | Pins that allow the device to connect and interact with external electronics such as LEDs, sensors, motors, and buttons.|
| USB | Used to connect peripherals like keyboards, mice, flash drives, and other USB devices.|
| HDMI |Sends video and audio signals to a monitor, TV, or projector for display. |
| Power |Supplies the electrical energy needed for the device to operate. |
| microSD |Provides storage for the operating system, applications, and user files. |
| Network |Enables wired internet and local network connectivity, typically through an Ethernet port. |
| Wireless |Provides cable-free communication through technologies such as Wi-Fi and Bluetooth. |
