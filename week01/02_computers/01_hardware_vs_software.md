# Part 1 – Hardware vs Software

## Explain Hardware
Write **at least 50 words** explaining what hardware is.
---Hardware is the physical part of a computer that you can actually see and touch. It includes devices such as the keyboard, mouse, monitor, processor, motherboard, speakers, and storage drives. Think of hardware as the body of a computer. Each component has a specific job, such as displaying information, storing files, or processing data. For example, when you type on a keyboard, the keyboard acts as hardware that sends information to the computer. Without hardware, a computer would not exist in a usable form because there would be no physical components to perform tasks or interact with users.

## Explain Software
Write **at least 50 words** explaining what software is.
---Software is the set of programs and instructions that tell a computer what to do. Unlike hardware, software cannot be touched because it exists digitally. Common examples include operating systems, web browsers, video games, and word-processing applications. Software can be thought of as the brain or intelligence of a computer system because it directs how the hardware should operate. For instance, when you open a web browser, the software provides the instructions needed to display websites and interact with online content. Without software, computer hardware would be unable to perform useful tasks because it would have no guidance or commands to follow.

## How Do Hardware and Software Interact?
Write **at least 50 words** describing how they interact.
Hardware and software depend on each other to make a computer work effectively. A simple way to understand their relationship is to compare a computer to a human body. Hardware is like the body, while software is like the mind that controls actions. When you click a mouse, the hardware sends a signal to the computer. The software then interprets that signal and tells the screen what to display. Similarly, when you save a document, the software instructs the storage hardware to keep the file safely. Every activity on a computer, from sending emails to watching videos, requires hardware and software to work together. Neither can function properly without the other, making their partnership essential for all computing tasks.
