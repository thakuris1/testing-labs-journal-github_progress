  110100

To find the two's complement of a binary number:

Find the one's complement (invert all bits: 0 ↔ 1).
Add 1 to the result.
1) Binary Number: 101010

One's Complement:

101010 → 010101

Add 1:

  010101
+      1
--------
  010110

Two's Complement: 010110₂

2) Binary Number: 110011

One's Complement:

110011 → 001100

Add 1:

  001100
+      1
--------
  001101

Two's Complement: 001101₂

3) Binary Number: 1001

One's Complement:

1001 → 0110

Add 1:

  0110
+    1
------
  0111

Two's Complement: 0111₂

Final Answers
Binary Number	Two's Complement
101010	010110₂
